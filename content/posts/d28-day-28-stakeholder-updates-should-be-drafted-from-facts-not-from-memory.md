# D28 - Day 28: Stakeholder updates should be drafted from facts, not from memory

Comms quality drops when update drafting starts from a blank page every time.
Day 28 drafts audience-specific updates from structured status and action context.

Problem: Operational updates are repetitive, inconsistent, and often miss critical context for different audiences.

MVP achieved: Comms drafter turns structured inputs into clear stakeholder messages with actions and next-step framing.

Scale path: Template governance, tone variants by audience, and approval workflow integration.
